## Template for Weekly Update

```markdown
## Week of [DATE]

**Completed:**
- [What got done this week]
- [Deliverables shipped]
- [Milestones reached]

**In Progress:**
- [What you're actively working on]
- [Percentage complete if applicable]
- [Expected completion date]

**Blockers:**
- [Anything blocking progress]
- [External dependencies]
- [Client decisions needed]

**Next Week:**
- [What you plan to complete]
- [Key actions]
- [Client touchpoints]

**Notes:**
- [General observations]
- [Client feedback or requests]
- [Anything unexpected]
```

**Update every Friday** with the current week's summary. This becomes your memory for:
- Retrospective (what happened this week?)
- Standup (what's the status for scheduling?)
- Client communication (evidence of progress for invoicing)
